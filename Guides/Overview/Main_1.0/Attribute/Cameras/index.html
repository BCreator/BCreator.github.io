    光源（组件）

    摄像机（组件）

    BCreator.io场景是通过在三维空间中排列和移动对象而创建的。由于查看器的屏幕是二维的，因此需要一种方法来捕获视图并将其“展平”以进行显示。这是通过使用摄像机完成的。

    摄像机是在场景空间中定义视图的对象。对象的位置定义了视点，而对象的向前（Z）和向上（Y）轴分别定义了视图方向和屏幕顶部。该摄像机组件还定义视图落入该区域的尺寸和形状。通过设置这些参数，摄像机可以在屏幕上显示当前“看到”的内容。随着摄像机对象的移动和旋转，显示的视图也将相应地移动和旋转。

    透视相机和正交相机

    透视模式（上）和正交模式（下）中显示的同一场景

    现实世界中的摄影机，或者实际上是人眼，以一种使物体从视点越远看起来越小的方式观看世界。这种众所周知的透视效果已广泛应用于艺术和计算机图形学中，对于创建逼真的场景非常重要。自然，BCreator.io支持透视摄像机，但是出于某些目的，您希望在不具有此效果的情况下渲染视图。例如，您可能想要创建一个不应该看起来像真实世界对象的地图或信息显示。不会随距离缩小对象大小的摄影机称为正交摄影机，BCreator.io摄影机也可以选择这种方式。观看场景的透视和正交模式称为摄像机投影。

    观看区域的形状

    透视摄像机和正交摄像机都限制了它们从当前位置可以“看到”的距离。该限制由垂直于相机向前（Z）方向的平面定义，这被称为远裁剪平面，因为距相机更远距离的对象被“裁剪”（即从渲染中排除）。在摄像机附近还有一个对应的近裁剪平面——可见的距离范围是两个平面之间的距离。

    如果没有透视图，则无论其距离如何，对象的大小都相同。这意味着，正交摄影机的观看量由在两个剪切平面之间延伸的矩形框定义。

    该矩形框使用透视时，随着距相机的距离增加，物体的尺寸似乎会减小。这意味着场景可见部分的宽度和高度会随着距离的增加而增加。因此，透视摄像机的观看体积不是盒子，而是金字塔形状，其顶点位于摄像机的位置，底部位于远的剪切平面 。但是，形状并非完全是金字塔形的，因为顶部被附近的剪切平面切掉了，这种截顶的金字塔形状称为平截头体。

    相机视图的背景

    对于室内场景，摄像机可能总是完全位于代表建筑物、洞穴或其他建筑物内部的某个物体内部。但是，当该动作发生在户外时，物体之间将有许多空白区域，这些区域什么也没填充。这些背景区域通常代表天空、空间或水下场景的模糊深度。

    相机无法完全确定背景，因此必须在空白处填充一些东西。最简单的选择是在将场景渲染到顶部之前，将背景清除为纯色。您可以从检查器使用相机的背景色属性设置此颜色。与室外场景配合使用的更复杂的方法是使用天空盒（组件）。顾名思义，天空盒的行为就像是一个排列有天空图像的“盒子”。相机有效地放置在此框的中央，可以从各个方向看到天空。相机旋转时会看到不同的天空区域，但其中心不会移动，因此相机无法“靠近”天空。天空盒渲染在场景中所有对象的后面，因此它代表了无限远处的视图。最常见的用法是在标准的室外场景中代表天空，但实际上，天空盒甚至完全包围了相机。这意味着您可以使用天空盒来表示场景的各个部分，例如，延伸超出地平线的起伏的平原、太空或水下场景的全方位视图。

    你可以简单地通过设置添加一个天空盒的场景（属性区：添加组件 →渲染 →天空盒）。请参阅此页面进一步了解如何创建自己的天空盒的细节。
