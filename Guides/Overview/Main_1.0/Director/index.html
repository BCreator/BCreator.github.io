    新建

    点击创建按钮，选择保存位置及文件名，然后进行编辑。

    导演行为树(Behavior Tree)

    导演行为树只有4大类型的Node节点：

    • Composite Node　　组合节点
    • Decorator Node　　 修饰节点
    • Condition Node　　 条件节点（叶节点）
    • Action Node　　　　动作节点（叶节点）

    Composite Node　　组合节点

    按顺序去执行：按（添加）顺序依次执行其子节点，只要其中某个子节点失败，该节点返回失败，不再继续执行下面的子节点。当所有子节点执行成功，该节点返回成功。

    选择（一个去）执行：按（添加）顺序依次执行其子节点，只要其中某个子节点执行成功，该节点则返回成功，不再继续执行下面的子节点。当所有子节点执行失败，该节点返回失败。

    并行节点：同时执行所有子节点，只要其中某个子节点失败，该节点返回失败，不再继续执行。当所有子节点执行成功，该节点返回成功。

    Decorator Node　　 修饰节点

    总是失败：执行子节点，返回失败。

    总是运行：执行子节点，返回运行。

    总是成功：执行子节点，返回成功。

    直到失败：循环执行子节点，直到子节点返回失败。

    直到成功：循环执行子节点，直到子节点返回成功。

    执行次数限制：循环执行子节点一定次数

    重复执行：无限循环执行子节点

    Condition Node　　 条件节点

    满足所有条件（与）：只要有一个条件的返回值为成功，或节点返回成功。所有条件子节点都返回为失败，或节点返回失败。

    总是失败：返回一个失败。

    满足任意一个条件（或）：只要有一个条件的返回值为失败，与节点返回失败。所有条件子节点都返回为成功，或节点返回成功。

    永远成功：返回一个成功。

    AIFire（AI开火）：配合AI寻找目标、AI随机移动，让AI向寻找到目标的位置发射炮弹。

    AIRandomMove（AI随机移动）：AI随机进行移动。

    AIUpdateEnemy（AI自动寻找目标）：AI寻找范围内不同阵营目标。

    MoveTo（移动到）：移动到指定坐标点。

    需要在属性区配置x、z坐标，默认为0，如下图所示：

    MoveToPosition（移动到位置）：移动以指定速度到指定坐标点。

    MoveWithDirection（向方向移动移动）：移动到指定坐标点。
