    资源管理

    资源管理位于引擎的左侧，用于添加项目相关资源。可通过拖动资源管理中的资源将其放置在场景中

    资源分类

    制作一款游戏，首先需要制定策划案，然后准备游戏资源，一个游戏项目里会有各种各样的资源，需要对资源进行合理的管理。

    资源管理最直观的体现在于对文件的归类与命名。在IO中，所有游戏相关文件都被放置在 Assets 文件夹下，如下图所示，常见文件夹的内容如下表所示。

    文件夹 内 容
    Models 模型文件，其中包括自动生成的材质球文件
    Prefabs 预制体文件
    Scene 场景文件
    Scripts 脚本代码文件
    Audio 音频文件
    Texture 贴图文件
    Behavior 导演树
    贴图 图片

    操作

    在资源管理中可以对项目资源进行创建、删除、重命名、导入、导出等操作。

    创建

    可创建文件夹、C#脚本、预制件、材质。在资源管理中创建的预制件可以在IO界面的资源库中看到使用。

    删除

    删除项目中的资源文件。

    重命名

    重命名项目中的资源文件。

    导入

    将本地的资源导入到当前项目中。

    导出

    将当前项目中的资源导出到本地。
