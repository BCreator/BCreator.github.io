    视图窗口

    视图窗口包括场景视图、游戏视图和导演视图。场景视图是对可视对象，进行编辑和展示的区域。游戏中所有关于美术、界面、文字等可视部分的编辑，都可以在场景视图上完成。游戏视图是从游戏中的相机渲染的。它代表了您最终发布的游戏。您将需要使用一个或多个Cameras以控制玩家在玩游戏时实际看到的内容。导演视图用来创建编辑导演树，该区域的操作，将在手册中的导演部分进行更加详细的介绍。

    场景窗口

    在场景窗口滚动鼠标滚轮可以快速拉远拉近视窗。按住鼠标滚轮可以让拖动视窗。按住鼠标右键可以调整视窗角度。

    添加，修改和删除对象 需要创建已有对象的新实例，可以在场景中选中需要新创建的实例，通过复制粘贴的方式，或者在资源库直接拖拽对象到场景中。

    修改实例位置和大小，可以在场景选中需要修改的实例，选中后实例周围会显示调整位置的边框，拖拽实例可以移动实例位置，旋转可以调整实例角度。

    删除实例，选中实例后按Delete键删除实例。删除实例不会从资源库中删除对象。

    视窗角度切换、网格

    在视图窗口右上方为视窗角度切换、网格按钮。

    点击视窗角度切换，选择顶底左右正背切换展示角度。

    点击网格按钮打开关闭网格显示。
